* 명령 프롬프트(CMD)에서 데이터 베이스 접속

ㅇ cmd 실행
ㅇ 유정 계정으로 로그인 방법  - cmd 
윈도우 로고키 + r   하고   cmd 입력하고 엔터
mysql 접속 방법 
mysql -u 계정명 -p    엔터
비밀번호 입력
(1/20) 계정명: javaUser
비번 1234

ㅇ show databases; 
데이터 베이스 출력

ㅇ use [데이터베이스명]
데이터베이스 사용

↑ 이 과정들을 거치고 나서 
select 든 delete든 명령어를 수행해야 내가 작업 혹은 입력한 데이터베이스에 접근이 가능함!
